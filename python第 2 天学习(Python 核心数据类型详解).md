### 模块 1：3 种核心数据类型（新手必懂）

Python 里所有数据都有「类型」，不同类型能干的事不一样，你后续写脚本 99% 只会用到这 3 种，也是第 1 天已经接触过的：

|类型|全称|说明|示例|
|---|---|---|---|
|`str`|字符串|用双引号`\&\#34;\&\#34;`/ 单引号`\&\#39;\&\#39;`包裹的内容，`input\(\)`获取的内容永远是字符串|`name = \&\#34;张三\&\#34;`、`url = \&\#34;https://github\.com\&\#34;`|
|`int`|整数|不带小数点的数字，用于数学计算|`birth\_year = 1994`、`port = 80`|
|`float`|浮点数|带小数点的数字，用于精确计算|`height = 1\.78`、`delay = 0\.5`|

#### 必学工具：`type\(\)` 查看变量类型

90% 的新手报错，都是因为「数据类型不对」，用`type\(\)`可以直接查看变量的类型，排查报错超级好用。
在 VS Code 的`day2\.py`里输入这段代码，按`Ctrl\+F5`运行：

```python
# 用type()查看变量的类型
name = "荔枝Python"
birth_year = 1994
height = 1.78

print(type(name))       # 输出 <class 'str'> 字符串类型
print(type(birth_year)) # 输出 <class 'int'> 整数类型
print(type(height))     # 输出 <class 'float'> 浮点数类型
```

#### 新手必踩的天坑（提前避坑）

**`input\(\)`****获取的所有内容，哪怕你输入的是数字，类型永远是字符串****`str`****！**
直接运行下面这段代码，你会看到报错：


报错原因：你输入的`1994`，在程序里是带引号的字符串`\&\#34;1994\&\#34;`，不是能计算的数字，这就是我们接下来要解决的问题。

### 模块 2：类型转换（核心必学）

把一种数据类型转换成另一种，解决上面的报错问题，也是写脚本时每天都会用到的功能，核心 3 个函数：

1. `int\(内容\)`：把符合数字格式的内容转换成整数

2. `str\(内容\)`：把任意内容转换成字符串

3. `float\(内容\)`：把符合数字格式的内容转换成浮点数

#### 1\. 解决上面的报错：input 输入数字计算

```python
# 正确写法1：分步转换
birth_year = input("请输入你的出生年份：")
birth_year = int(birth_year) # 把字符串"1994"转成数字1994
age = 2026 - birth_year
print("你的年龄是：", age)

# 正确写法2：一行搞定（推荐日常用）
birth_year = int(input("请输入你的出生年份："))
age = 2026 - birth_year
print("你的年龄是：", age)
```

#### 2\. 字符串和数字拼接：用 str \(\) 转换

```python
age = 32
# 错误写法：print("我的年龄是"+age) 字符串不能直接+数字
# 正确写法：用str()把数字转成字符串
print("我的年龄是："+str(age))
```

#### 后续使用场景

- 爬虫：拼接分页网址`\&\#34;https://xxx\.com/page/\&\#34;\+str\(page\_num\)`

- 网安：拼接 IP 和端口`ip\+\&\#34;:\&\#34;\+str\(port\)`

- 自动化：把命令执行的数字结果转成文本保存

### 模块 3：字符串高频操作（爬虫 / 网安必备）

字符串是你后续写脚本用得最多的类型：处理网址、爬取的文本、设备命令输出、请求头内容，全靠字符串操作。今天只学最高频、必用的 5 个，学完马上能落地。

#### 1\. 字符串拼接：用`\+`号

把多个字符串连在一起，拼接网址、输出文本必用

```python
# 字符串拼接
first_name = "Yan"
last_name = "618"
full_name = last_name + first_name
print(full_name) # 输出618Yan

# 带提示文字拼接
print("你的完整昵称是：" + full_name)
```

⚠️ 注意：`\+`号拼接时，所有内容必须都是字符串，数字要先用`str\(\)`转换。

#### 2\. 字符串切片：提取指定内容

从长字符串里提取你想要的部分，爬虫提取网址里的用户名、网安提取 IP 里的网段，超级常用。
**语法**：`字符串\[起始索引:结束索引\]`

- Python 索引从`0`开始，不是 1

- 左闭右开：包含起始位置的内容，不包含结束位置的内容

```python
# 切片示例（用你的GitHub地址举例）
url = "https://github.com/daloyanf/python-.git"

# 提取网址里的用户名
user_name = url[19:27]
print(user_name) # 输出daloyanf

# 常用简写
print(url[:8]) # 从开头取到第8位，输出https://
print(url[19:]) # 从第19位取到结尾，输出daloyanf/python-.git
```

#### 3\. `strip\(\)`：去掉首尾多余空格 / 换行

用户输入时经常会不小心打空格，爬虫爬下来的内容也会有多余空白，这个函数能一键清理，必用！

```python
# strip() 去掉首尾空格
name = input("请输入你的名字：") # 用户不小心输入了  张三  （前后有空格）
name = name.strip() # 一键去掉前后空格
print("你好，"+name)

# 进阶：input的时候直接处理
name = input("请输入你的名字：").strip()
```

#### 4\. `lower\(\)`/`upper\(\)`：大小写转换

处理网址、请求头、用户输入时，避免大小写不一致导致的问题，比如`HTTPS`和`https`在程序里是两个不同的内容。

```python
# 大小写转换
url = "HTTPS://GITHUB.COM"
url = url.lower() # 转成全小写
print(url) # 输出https://github.com

name = "yan"
name = name.upper() # 转成全大写
print(name) # 输出YAN
```

#### 5\. `replace\(\)`：替换指定内容

爬虫里替换掉文本里的无效字符、自动化里修改命令参数，超级常用。
**语法**：`字符串\.replace\(要替换的内容, 替换成的内容\)`

```python
# replace() 替换内容
text = "我的出生年份是1994"
new_text = text.replace("1994", "1995")
print(new_text) # 输出我的出生年份是1995

# 爬虫场景：去掉文本里的换行符
content = "这是爬取的\n文本内容"
content = content.replace("\n", "")
print(content) # 输出这是爬取的文本内容
```
