## 00:15–00:40 【核心知识点】逐行讲解+VS Code实时实操

每学一个知识点，就在`day1.py`里写对应代码，写完右键运行，看终端输出，确保每一行都懂。

### 1. print() 函数：程序的“输出嘴巴”

**作用**：在终端里打印你想显示的内容，是Python最基础的语句

```Python

# 这是注释：# 后面的内容是给人看的，程序不会执行，VS Code里会变灰色
print("我的第一个Python程序")  # 打印文字，必须用双引号""或单引号''包起来
print(123)  # 打印数字，不用加引号
print("今天是学习的第1天")
```

**VS Code实操提示**：写完3行，右键运行，看终端会不会依次输出3行内容。

**补充技巧**：按 `Ctrl+/` 可以快速给选中的行加注释/取消注释，写代码必备。

### 2. 变量：存储数据的“盒子”

**作用**：把你要用的数据存起来，方便后续重复使用，是所有代码的基础

**语法**：`变量名 = 你要存的数据`

```Python

# 给变量赋值，把数据放进“盒子”里
name = "张三"  # 存文字（字符串类型）
age = 20  # 存整数（数字类型）
height = 1.78  # 存小数（浮点数类型）

# 用print打印变量里存的内容
print(name)
print("我的年龄是：", age)  # 逗号可以同时打印文字和变量，中间会自动加空格
```

**关键解释**：

- `=` 是「赋值」，不是数学里的“等于”，意思是把右边的数据，放进左边的变量盒子里

- 变量名可以自己起，建议用英文，见名知意，不要用中文、数字开头、特殊符号

- 变量可以重复赋值，新的值会覆盖旧的值

### 3. input() 函数：程序的“输入耳朵”

**作用**：让程序停下来，等待你在终端里输入内容，实现和用户的交互

**语法**：`变量 = input("给用户的提示文字")`

```Python

# 运行后，程序会在终端显示提示文字，等你输入，输完按回车，内容会存进name变量里
name = input("请输入你的名字：")
age = input("请输入你的年龄：")

# 打印你输入的内容
print("你好，", name)
print("你的年龄是：", age)
```

**VS Code必看提示**：

运行这段代码后，终端会出现`请输入你的名字：`，光标会一直闪，**这不是程序卡住了**，是在等你输入！

你要点击终端面板，输入你的名字，按回车，再输入年龄，再按回车，程序才会继续执行。

### 4. 简单数学运算

Python可以直接当计算器用，后续爬虫、网安脚本里会经常用到数字处理

```Python

a = 10
b = 3

print(a + b)  # 加法，输出13
print(a - b)  # 减法，输出7
print(a * b)  # 乘法，输出30
print(a / b)  # 除法，输出3.3333333333333335
```

### 0基础常见报错&解决方法

1. **报错：** **`python 不是内部或外部命令`**

原因：安装Python时没勾Add Python to PATH，或者没选对解释器

解决：重装Python勾上PATH，重新按步骤选对解释器

1. **报错：** **`NameError: name 'xxx' is not defined`**

原因：变量名写错了，或者没定义就直接用了

解决：检查变量名拼写，确保先赋值再使用

1. **运行后终端没反应/卡住**

原因：代码里有`input()`，程序在等你在终端输入内容

解决：点击终端面板，输入对应内容，按回车即可

1. **改了代码，运行还是旧内容**

原因：没保存文件

解决：每次改完代码，按`Ctrl+S`保存，再运行
